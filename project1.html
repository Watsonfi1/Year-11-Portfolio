End Users

This project is targeted towards a employer that needs 3D character models created for a video game or other project. They would look at this work to see what I have done and how I could help them create models. They will need to see that I can create a good 3D character and that I'm able make animations. Game developers might also want to use the model and animations I have made for their game. My target is countries that speak English. Employers might want to know what software I used and what skills were used in the creation of my character and animations.

Implications



Aesthetics

Aesthetics on a low poly character is the way that the character looks. Aesthetics make a character look either appealing or non-appealing.

I addressed this by:

  • I will make a human character be wearing an outfit, hat and maybe a beard or other facial feature.
  • I will make the character tall and skinning with big eyes.

Functionality

Functionality of a low poly character is the rigging. Making sure that your characters arms, legs, back and neck are able to move.

I addressed this by:

  • I will make sure that my character has good rigging across the body that looks normal when it moves. I will also make sure that all the nessersary rigging point plus more are done well.
  • I will make sure the animations work well and also look natural. I will do this by testing everything until it looks normal and good.

Usability

Usability of a low poly character would be that the character moves smoothly and has a low polygon count so games won't lag.

I addressed this by:

  • I will test the characters animations lots to make sure that the character is moving smoothly and isn't glitching.
  • I will make sure that all the polygons used in the character are needed to make it the least laggy possible in a game. I will check using the see through mode where you can see all the vertices.

Intellectual Property

Intellectual property is making sure that my character isn't copying anybody else intellectual property.

I addressed this by:

  • I will make sure I think up of the character my self and any thing I use as a base doesn't look to simular to my character.
